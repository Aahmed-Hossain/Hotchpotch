# To Run on the local m/c.
```bash
git clone https://github.com/Aahmed-Hossain/Hotchpotch.git
cd Hotchpotch
npm install
npm run dev
```
## Tech Stack
-Next.js
-TypeScript
-Tailwindcss
-Material Ui
